# 游戏常量配置表

> 文件位置：`src/constants.js`
> 修改后需重启 dev server 生效。

---

## 一、游戏规则常量

| 常量 | 当前值 | 类型 | 说明 | 调参影响 |
|------|--------|------|------|---------|
| `TRACK_CAP` | `5` | 整数 | 轨道最大同时存在的炮台数 | 越小越难（玩家调度压力增加） |
| `BUFFER_CAP` | `5` | 整数 | 暂存区初始容量；道具一最多扩展到 `BUFFER_CAP+3=8` | 越小越难（溢出更早触发） |
| `TURRET_SPEED` | `3` | px/帧 | 炮台沿轨道的移动速度；冲刺时乘以 `speedMult`（最大 2.4） | 越大越快，射击窗口越短 |
| `BULLET_SPEED` | `14` | px/帧 | 子弹飞行速度 | 越大命中越快，视觉反馈越短 |

## 二、布局常量

| 常量 | 当前值 | 说明 | 备注 |
|------|--------|------|------|
| `VW` | `480` | 视口宽度（px） | 修改需同步 main.js Phaser config |
| `VH` | `920` | 视口高度（px） | 修改需同步 main.js Phaser config |
| `TRACK_GAP` | `22` | 轨道到画布边缘的间距（px） | 影响炮台运动路径和画布视觉间距 |
| `CELL_MIN` | `6` | 格子最小像素尺寸 | 大网格关卡的格子下限 |
| `CELL_MAX` | `18` | 格子最大像素尺寸 | 小网格关卡的格子上限；特效缩放基准值也是 18 |

## 四、冲刺机制参数

冲刺触发和加速逻辑在 `GameLogic._checkEndgame` 和 `_handleLapComplete` 中硬编码，当前参数：

| 参数 | 当前值 | 说明 |
|------|--------|------|
| 触发阈值 | `bufferCap + 1`（默认 6） | 剩余炮台总数低于此值时触发冲刺 |
| 初始速度倍数 | `2.0` | 冲刺触发时 `speedMult` 设为此值 |
| 每圈加速倍率 | `1.2` | 每完成一圈 `speedMult *= 1.2` |
| 速度上限 | `2.4` | `speedMult` 最大值 |

---

## 五、道具系统参数

道具次数和上限在 `ItemSystem` 中硬编码，当前参数：

| 道具 | 每关次数 | 效果上限 | 说明 |
|------|---------|---------|------|
| 道具一（＋槽） | `3` | `BUFFER_CAP + 3 = 8` | 每次 `bufferCap++` |
| 道具二（取车） | `3` | 无 | 从队列任意位置强制部署，忽略 `trackCap` |
| 道具三（清色） | `3` | 无 | 清除指定颜色所有方块+炮台，清色后触发冲刺检测 |

---

## 六、渲染深度（Depth）层级

| 层 | depth 值 | 内容 |
|----|---------|------|
| 主画布 | `0` | 网格、轨道、炮台、子弹、粒子、暂存区、队列 |
| 文本 | `6` | 炮台弹药、队列数量等文本池 |
| 道具图标 | `8~9` | 道具圆圈 + 文字 |
| 胜负遮罩 | `10` | overlayG |
| 胜负文字 | `11` | txStatus / txContinue |
| 道具特效 | `18~21` | 暗角、聚光、轮盘、飞行动画 |
| DevTools 面板 | `99` | 开发跳关面板 |
| DevTools 按钮 | `100` | 开发跳关按钮 |
| DevTools 遮罩 | `102` | 吞掉面板隐藏时的穿透点击 |
