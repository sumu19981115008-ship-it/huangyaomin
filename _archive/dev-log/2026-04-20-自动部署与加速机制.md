# 自动部署与加速机制开发记录

**日期：** 2026-04-20  
**分支：** master  
**涉及文件：** `src/GameLogic.js` / `src/GameScene.js` / `src/constants.js`

---

## 功能描述

当剩余炮车总数满足条件时，自动触发"终局加速模式"：
- 等待区（lanes）+ 暂存区（buffer）的所有炮车按 0.3 秒间隔依次自动进入转盘
- 触发时所有在轨炮车立即提速到基础速度的 2 倍
- 此后每完成一圈提速 ×1.1，上限为基础速度的 2.88 倍
- 触发后炮车不再进入暂存区，子弹打光前持续旋转
- 该模式只触发一次，不重复触发

---

## 触发条件（三个条件同时满足）

| 条件 | 说明 |
|------|------|
| `autoDeployTriggered === false` | 未曾触发过 |
| `total > 4` | 关卡炮车总数超过4辆（小关卡不触发） |
| `total <= BUFFER_CAP`（≤5） | 转盘+暂存区+队列炮车总数不超过暂存上限 |

`total = turrets.length + buffer.length + lanes 所有炮车之和`

## 核心改动

### 1. `ActiveTurret` 新增 `speed` 字段

```javascript
constructor(color, ammo, speed = TURRET_SPEED) {
  this.speed = speed;
  // ...
}
```

每帧移动改为 `t.pathPos += t.speed`（原为固定常量 `TURRET_SPEED`）。

### 2. `onLapComplete()` 每圈加速

```javascript
onLapComplete() {
  const max = TURRET_SPEED * 2.88;
  this.speed = Math.min(max, this.speed * 1.1);
}
```

### 3. `activateAutoDeploy()` 触发入口

```javascript
activateAutoDeploy() {
  this.autoDeployTriggered = true;
  for (const t of this.turrets) {
    t.speed = TURRET_SPEED * 2; // 已在轨道上的炮车立即提速
  }
}
```

### 4. `_handleLapComplete()` 触发后持续旋转

```javascript
if (this.autoDeployTriggered && t.ammo > 0) {
  t.pathPos -= TOTAL_DIST; // 保留误差，位置平滑衔接
  t.lapComplete = false;
  t.shotSlotsThisSide = new Set();
  t.lastSide = SIDE.BOTTOM;
  return; // 不进 buffer，不移除，继续旋转
}
```

### 5. 新入场炮车以 2 倍速入场

`forceDeployFromBuffer` 和 `forceDeployFromLane` 均以 `TURRET_SPEED * 2` 初始速度创建/重置炮车。

### 6. `shouldTriggerAutoDeploy()` 触发检测

```javascript
shouldTriggerAutoDeploy() {
  if (this.autoDeployTriggered) return false;
  const pending = this.buffer.length + this.lanes.reduce((s, l) => s + l.length, 0);
  const total = pending + this.turrets.length;
  return total > 4 && total <= BUFFER_CAP;
}
```

### 7. `GameScene._checkAutoDeploy()` 定时部署

每帧检测触发条件，触发后：
1. 调用 `logic.activateAutoDeploy()`
2. 对 buffer（优先）和 lanes 中所有炮车，用 `delayedCall` 按 **0.3 秒间隔**依次强制入场

## 速度设计参数

| 参数 | 值 |
|------|----|
| 基础速度 `TURRET_SPEED` | 3 |
| 触发时初始速度 | `3 × 2 = 6` |
| 每圈加速倍率 | ×1.1 |
| 最高速度上限 | `3 × 2.88 = 8.64` |
| 从触发速度到上限所需圈数 | ≈4圈（6 × 1.1^4 ≈ 8.78，3圈时 ≈7.99） |

## 行为对比

| 场景 | 触发前 | 触发后 |
|------|--------|--------|
| 炮车速度 | 固定 `TURRET_SPEED=3` | 立即升至 6，每圈 ×1.1，上限 8.64 |
| 圈完成有剩余子弹 | 进入暂存区（buffer） | 继续旋转 |
| 圈完成子弹打光 | 从转盘移除 | 从转盘移除（同） |
| 暂存区溢出判定 | 正常判定 | 不再触发溢出失败 |
| 新入场炮车速度 | `TURRET_SPEED=3` | `TURRET_SPEED × 2 = 6` |
